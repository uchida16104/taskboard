Visualize ideas as tasks.

Usage

  1. Copy and paste markdown from the AI Prompt into each LLM.
  2. Create or transcribe output results into new discussions for Ideas.
  3. Select one label: “approved”, “rejected”, or “needs-review”.

First time: set these labels properly.

Discussion Behavior

  • Closed when “rejected”.
  • No change when “needs-review”.
  • Opened when “approved”.

Projects number and views number optional.
