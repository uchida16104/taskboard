# Task　Board
Visualize ideas as tasks.

## Usage
1. [Create new discussions for Ideas](https://github.com/Uchida16104/TaskList/discussions/new?category=ideas)
2. Please select one of the following labels: “approved”, “rejected”. or “needs-review”. <br />***(Please set to add these label for the first time.)***
3. The discussion will be, 
* closed when you select "rejected". 
* done nothing when you select "needs-review".
* opened when you select "approved", and please go to https://github.com/users/name/projects/${projectsNumber}/views/${viewsNumber}.<br />
***${projectsNumber} and ${viewsNumber} are set optional.***

4. Going https://github.com/users/name/projects/${projectsNumber}/views/${viewsNumber}, please create the following columns.  <br />***(Please set this for the first time.)***

| Column Name |
| ------- |
| Backlog |
| Todo |
| Doing |
| Done |

5. Please move task tickets according to their progress status.
